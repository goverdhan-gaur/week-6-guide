# EMERGENCY TROUBLESHOOTING

## ADS NOT DELIVERING

**Check:**
1. Account spending limit (Meta Business Settings)
2. Payment method valid
3. Ads approved (check Ads Manager status)
4. Budgets sufficient ($50+ minimum)

**Fix:**
- Increase spending limit
- Update payment method
- Edit rejected ads
- Increase budgets

---

## NO SALES (ADS RUNNING BUT 0 CONVERSIONS)

**After $200 spend with 0 sales:**

**Check:**
1. Pixel firing? (use Facebook Pixel Helper)
2. Clicks happening? (CTR > 1%?)
3. Add-to-carts? (check Shopify analytics)

**Diagnose:**
- No clicks = Bad creative (remake ad)
- Clicks but no ATCs = Bad landing page
- ATCs but no purchases = Checkout friction (shipping cost? payment issues?)

**Fix:**
- Bad creative â†’ New ad with different image/copy
- Bad landing page â†’ Simplify, add trust badges
- Checkout friction â†’ Free shipping, trust signals

---

## EMAIL NOT SENDING

**Check:**
1. Klaviyo connected to Shopify?
2. Email flow active (not draft)?
3. Segment not empty?
4. SPF/DKIM authenticated?

**Fix:**
- Reconnect integration
- Set flow to "Live"
- Check segment size
- Contact Klaviyo support for authentication

---
